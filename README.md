## 1. 📋 프로젝트 개요 (Overview)

### **배경 (Background)**
* 최근 인테리어 시장은 단순히 '예쁜 공간'을 소비하는 것을 넘어, 개인의 취향과 라이프스타일을 공간으로 표현하는 방향으로 변화하고 있습니다.
* 오늘의집의 현행 추천 시스템은 사용자의 **과거 행동 데이터를 전제**로 작동하기 때문에, 행동 이력이 없는 **신규 유저에게는 개인화 추천이 불가능**하다는 한계가 있습니다.
* EDA를 통해 가격이나 성능보다 '기분'과 '감정적 만족'을 최우선으로 고려해 지출하는 소비 트렌드, **필코노미(Feel+Economy)** 를 확인하고 사용자의 공간 인식 방식과 감성에 맞는 개인화 추천의 중요성에 집중하였습니다.

### **목표 (Goal)**
* 집들이 게시글 **8,000개**와 상품 태그 **194만 건**을 수집·분석하여, 사용자의 거주 조건과 공간 인식 유형에 기반한 **맞춤형 인테리어 상품 조합 추천 시스템**을 구현합니다.
* 행동 데이터 없이도 즉시 추천이 가능한 **Cold Start 문제 보완**에 초점을 맞추었습니다.

## 3. 🔍 분석 과정 (Pipeline)

### **① 데이터 수집**
* Selenium 기반 크롤링 4단계 (세션 인증 → 게시글 ID → 상세 정보 → 카테고리)
* 게시글 8,000건 / 상품 194만 건 / RGB 이미지 데이터 수집

### **② 전처리**
* RGB → HSV 기반 톤 변환 (5개 레벨 × 6개 색상 계열, 최대 30종)
* 평수 구간화 / 공간 레이블 22→13개 정제 / 예산 피처 생성

### **③ 인테리어 트렌드 분석**
* 지역별 EDA / 상품별 EDA / 스크랩 전환율 분석
* 시계열 스타일 트렌드 분석 (2022~2025)
* 오늘의집 북촌 쇼룸 답사를 통한 트렌드 검증

### **④ 공간 인식 유형 분석 (NLP)**
선행연구(채혜진·오창섭 2019, 홍지수·최정민 2025, 오동근 2024)를 바탕으로 공간 인식 유형을 4가지로 정의하였습니다.

| 유형 | 설명 |
|---|---|
| **R, REST (휴식계열)** | 집을 휴식과 회복의 공간으로 인식하며 편안함과 힐링을 추구 |
| **T, TASTE (취향계열)** | 집을 취향과 개성을 표현하는 공간으로 인식하며 스타일과 감성을 중시 |
| **F, FAMILY (관계계열)** | 가족·타인과의 교류를 고려한 공간 활용 (아이방, 반려동물, 손님 맞이 등) |
| **H, HOBBY (작업계열)** | 취미와 작업을 위한 공간 중시 (서재, 작업실, 식물 공간 등) |

* KLUE-BERT, KLUE-RoBERTa, KcELECTRA, KoELECTRA 4개 모델 비교 실험
* **KLUE-BERT F1 Macro 0.872**로 최고 성능 기록 → 최종 모델 선정
* 가설검정(카이제곱)을 통해 인테리어 스타일 × 공간 인식 유형, 상품 조합 × 공간 인식 유형이 통계적으로 유의하게 다름을 확인

### **⑤ 상품 조합 추천 모델 (KNN)**
신규 유저의 Cold Start 문제를 해결하기 위해 KNN 기반 추천 파이프라인을 설계하였습니다.

* **피처 구성 (총 66개):** 스타일 12개, 색조 5개, 보조색조 5개, 공간 인식 유형 17개, 공간별 예산 13개, 공간 등장 여부 13개, 주색조 비율 1개
  * ※ 거주유형·평수·가족구성·시공방식·공사유형은 하드 필터로 사전 처리 (KNN 피처 미포함)
* **K=20** 기본값 확정 / 희소 조건 시 K=50 자동 확장
* **Fallback 로직:** 하드 필터 4단계 완화 + 카테고리 depth3 → depth2 → depth1 확장
* **최종 점수:** 등장빈도 40% + 평균 인기도 60% 가중합

**모델 검증:**
* 조건 변경 시 자카드 유사도 평균 0.035 → 조건별 실질적으로 다른 추천 확인
* 베이스라인(단순 인기도 순) 대비 97.6% 다른 상품 추천 / 두 모델 결과 전혀 겹치지 않음

### **⑥ 대시보드 구현**
Streamlit으로 구현하였으며 네 가지 주요 기능으로 구성됩니다.

* **공간 인식 유형 테스트:** 27문항 → R/T/F/H 유형 도출 → 추천 소프트 필터 자동 반영
* **맞춤 상품 추천:** 방전체 / 특정공간 / 단일상품 세 가지 모드
* **그리드 기반 배치:** 8,000개 게시글 이미지 분석 기반 공간별 상품 위치 결정
* **GPT 기반 이미지 합성:** GPT-4o Vision으로 상품 설명 및 공간 위치 파악 → gpt-image-1로 최종 합성
